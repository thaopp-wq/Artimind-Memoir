## Hướng dẫn sử dụng trên Google Flow

### Bước 1: Gen Storyboard

1. Mở Google Flow project
2. Tạo **12 scenes** (hoặc 12 image generations)
3. Paste từng prompt ở trên vào từng scene
4. Gen → review → chỉnh sửa nếu cần
5. Export từng frame ra ảnh

### Bước 2: Thêm Text Overlay

Dùng tool edit (Canva, Figma, hoặc ngay trong Flow nếu hỗ trợ):
- Font title: **Cormorant Garamond Bold** (matching app)
- Font step label: **DM Sans SemiBold**
- Màu text: #FFFFFF
- Step number: #0090FF
- Background overlay: #000000 60% (vùng text)

### Bước 3: Ghép thành Video

| Frame | Duration | Transition |
|-------|----------|------------|
| 1. Hook | 3s | Fade in |
| 2. Mở app | 2s | Cut |
| 3. Home | 2s | Cut |
| 4. Moments | 2s | Cut |
| 5. Template | 3s | Cut |
| 6. Upload photos | 4s | Cut |
| 7. Voice question | 3s | Cut |
| 8. Audio upload | 3s | Cut |
| 9. Script | 4s | Cut |
| 10. Processing | 4s | Cut |
| 11. Result | 7s | Slow fade in |
| 12. CTA | 3s | Fade in/out |
| **Total** | **40s** | |

### Bước 4: Thêm nhạc + âm thanh

- Nhạc: Piano nhẹ, emotional ambient, royalty-free
- Scene 11 (Result): Fade in giọng nói (voiceover mẫu hoặc audio demo)
- Volume nhạc giảm ở Scene 11, tăng lại ở Scene 12
